# Analyzing Customer Reviews on Apparel 
Introduction to AI, Carnegie Mellon University

Pipeline:

* Dataset: https://amazon-reviews-2023.github.io/
  * Download review and meta from Amazon_Fashion
* Install requirement packages
  
  ```bash
  pip install -r requirements.txt
     ```
* Run `data.ipynb` for data cleaning and visualization and to get `balanced_data.csv`
* Run `3 models without limiting topics.ipynb` for the first modeling approach
* Run `linear_rf_regression_xgboost.ipynb` for the second modeling approach
